General Questions

What is the current North Carolina Building Code based on?

The current North Carolina Building Code is based on the 2018 International Building Code (IBC) with North Carolina-specific amendments.

How often is the North Carolina Building Code updated?

The North Carolina Building Code is typically updated on a six-year cycle. The next major update is expected to be based on the 2024 IBC.

Can local jurisdictions amend the state building code?

No. In North Carolina, local jurisdictions must enforce the state building code without reducing its requirements. However, they may adopt more stringent requirements in some circumstances.

Permitting Questions

When do I need a building permit?

Generally, you need a building permit for new construction, additions, renovations that involve structural changes, and changes in occupancy classification. Minor repairs might be exempt.

How long does it take to get a building permit?

The time varies by jurisdiction and project complexity. Simple projects might take a few days, while complex commercial projects could take several weeks.

Code Compliance Questions

What is the difference between the Building Code and Existing Building Code?

The Building Code primarily applies to new construction, while the Existing Building Code provides alternative approaches for renovating, altering, repairing, or changing the occupancy of existing buildings.

How do I appeal a code official's decision?

You can appeal to the local Board of Appeals if one exists, or to the North Carolina Building Code Council. Appeals must typically be filed within a specific timeframe after the decision.
