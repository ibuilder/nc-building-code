Introduction

The North Carolina Building Code comprises a set of regulations that establish minimum requirements for buildings to ensure public safety, health, and welfare. These codes are based on the International Code Council (ICC) codes with North Carolina-specific amendments.

History

North Carolina began adopting statewide building codes in the 1930s. The current North Carolina Building Code is based on the 2018 International Building Code (IBC) with state-specific amendments. The NC Building Code Council is responsible for adopting and amending the building code.

Code Structure

The North Carolina Building Code consists of several volumes:

  • Building Code
  • Residential Code
  • Existing Building Code
  • Plumbing Code
  • Mechanical Code
  • Fuel Gas Code
  • Electrical Code
  • Fire Prevention Code
  • Energy Conservation Code

Code Adoption Process

The North Carolina Building Code Council reviews and adopts new code editions approximately every six years. The process includes public hearings, technical committee reviews, and a formal adoption procedure. Local jurisdictions must enforce the state-adopted codes without amendments that reduce the requirements.
